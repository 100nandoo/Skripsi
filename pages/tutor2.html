Mengendalikan Perangkat Kelistrikan

  1. Arahkan mouse ke arah Navbar
  2. Klik tombol Peralatan
  3. Kemudian akan muncul halaman seperti gambar di bawah



  4. Kolom sebelah kiri merupakan berbagai tombol untuk mengendalikan perangkat kelistrikan. Kolom sebelah kanan merupakan berbagai tombol peralatan lainnya. Seperti phpMyAdmin.
  5. Tekan salah satu tombol jika ingin menyalakan/mematikan perangkat tersebut. Sebagai contoh, jika ingin menyalakan LED1 tekan tombol LED1. Jika ingin mematikan LED1 maka tekan lagi tombol LED1.
